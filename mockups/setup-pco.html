ServiceTracks reads your upcoming service plans from PCO so it knows which songs to sync. You'll create a Personal Access Token and paste it here — it takes about two minutes.

Create a Personal Access Token

  1. 1 Open your PCO API settings ↗
  2. 2 Click New Personal Access Token
  3. 3 Give it any description (e.g. "ServiceTracks").
  4. 4 Copy the Application ID and Secret into the form.

Your credentials are encrypted.

Tokens are stored with Fernet encryption. We only read your service plans and songs.
